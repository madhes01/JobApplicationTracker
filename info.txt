# Prerequisites (install these manually)
# - Node.js: https://nodejs.org
# - pnpm: npm install -g pnpm
# - Git: https://git-scm.com

# 1. Clone your project
git clone your_repo_url
cd your_project_folder

# 2. Install dependencies
pnpm install

# 3. Create .env file and add Neon connection string
# Add this line to .env:
# DATABASE_URL=your_neon_connection_string

# 4. Create .env.local and add these:
# DATABASE_URL=your_neon_connection_string
# BETTER_AUTH_SECRET=your_secret
# BETTER_AUTH_URL=http://localhost:3000
# NEXT_PUBLIC_BETTER_AUTH_URL=http://localhost:3000

# 5. Push Prisma schema to Neon
pnpm prisma db push

# 6. Generate Prisma client
pnpm prisma generate

# 7. Run Better Auth migration
pnpm dlx @better-auth/cli@latest migrate --config lib/auth/auth.ts

# 8. Start dev server
pnpm dev
Important Notes

pg and @types/pg are already in package.json so pnpm install handles them automatically — no need to install separately
Both .env and .env.local need DATABASE_URL — Prisma reads .env, Next.js reads .env.local
Never push either .env or .env.local to GitHub — keep connection string saved in Notion/Notes
